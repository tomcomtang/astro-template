# Markdown Example

This is a **Markdown** file demonstrating Astro's built-in support for `.md` files.

## Advanced Usage

Markdown files can be enhanced with:

- **Frontmatter**: Metadata for your content
- **Astro Components**: Embed Astro components in your Markdown
- **Styling**: Apply CSS classes and custom styles
